# Sensiya - Landing Page Audit Service

[Sensiya Deploymentation](https://vercel.com/fahad-x/sensiya/G31YpZyCiy1nPSZnjYCVYY3knYLB)

## Description

Sensiya is a high-converting landing page for a landing page audit service. We help early-stage SaaS companies identify what's killing their conversions and fix it.

## Tech Stack

- Vite
- React
- Tailwind CSS
- Lenis JS
- GSAP

## Features

- Fully responsive design (mobile, tablet, desktop)
- Dark theme with red accent color (#D12B27)
- Smooth animations and micro-interactions
- Mobile-first approach
- Optimized for conversion

## Sections

- Navbar section
- Hero section
- Problem section
- Solution section
- Analyze section
- Process section
- Testimonial section
- Pricing section
- Newsletter section
- Faq section
- Footer section
